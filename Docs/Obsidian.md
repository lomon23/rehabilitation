

# Налаштування обсідіана

Ми працюємо через абсідіан, тому потрібно скачати один плагін, та мати встановлений Git на пристрої.

--- 
## Порядок дій:
### Відкрити проєкт в Obsidian
* Склонувати репозиторій ```git clone https://github.com/lomon23/rehabilitation.git ```
* Відкрити цю дерикторію **Open folder as vault -> folder_of_repository
---
### Встановлення плагіна 
* Відкрити налаштування
* **community plugins ->Turn on community plugins -> browse
* В пошуку Git, автор Vinzent. -> install
* Перейти в список плагінів і включити плагін
---
### Налаштування автоматики
* Перейти в налаштування плагіна 
* Найти **Auto commit-and-sync interval** та поставити цифру 10
* Найти **Pull on startup** та ввімкнути
---
### Як працювати 
* можна просто працювати з текстом (створювати файли, редагувати, малювати схеми)
* Кожні 10 хв плагін автоматично комітить всі зміни та відправляє на сервер
* щоб відправити зміни прямо зараз потірбно **ctrl + P** або **command + P** . Ввести -> Git: commit -> Git: push
* Якщо просто читати файли, можна відкрити репозиторій ```https://github.com/lomon23/rehabilitation#``` та в папці Docs найти актуальні файли

## Примітки
* Старатись радагувати файли лише які виділені під тебе, або канвас, коли знаєш що ми не працюємо там одночасно, це для того щоб не було помилок синхронізації
