# Math & Coordinate Filtering

**Статус:** Draft

## 1. Фільтрація шумів (Anti-Jitter)
Сирі дані з MediaPipe містять мікро-флуктуації. Для стабілізації скелета використовується фільтр EMA (Exponential Moving Average).

- Кожна нова координата не застосовується миттєво. Вона "змішується" з попереднім станом.
- **Коефіцієнт Alpha (`a`):** Налаштовується динамічно (наприклад, 0.5). Нижче значення = більша плавність, але вища затримка.
- **Логіка:** `New_X = (a * Current_X) + ((1 - a) * Previous_X)`

## 2. Розрахунок кутів (Vector Math)
Бізнес-логіка правильного виконання вправи базується на вимірюванні кутів між суглобами.

- Визначаються три точки: A (початок), B (вершина кута/суглоб), C (кінець).
- Формуються два вектори: BA та BC.
- Кут обчислюється через арккосинус скалярного добутку векторів, поділеного на добуток їхніх довжин.
- Результат переводиться з радіанів у градуси `(radians * 180.0 / PI)`.

## 3. Валідація видимості (Confidence Score)
Алгоритм відкидає обчислення, якщо показник `visibility` для будь-якої з трьох точок падає нижче `0.65` (65%). У такому разі кадр ігнорується, щоб уникнути хибних результатів.
