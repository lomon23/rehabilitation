## Ізоляція потоків
Для забезпечення 30+ FPS без блокування інтерфейсу користувача, ML-модуль повністю винесено в ізольований потік.

1. **Main Thread (React):**
   - Захоплює кадр з `<video>` тегу через `requestAnimationFrame`.
   - Конвертує кадр у `ImageBitmap` (або `OffscreenCanvas` для економії пам'яті).
   - Відправляє об'єкт у Web Worker через `postMessage()`.
   - Отримує назад готовий JSON з координатами і малює `<canvas>`.

2. **Web Worker (TypeScript + MediaPipe):**
   - Ініціалізує `PoseLandmarker` з MediaPipe WebAssembly.
   - Синхронно проганяє отриманий кадр через модель.
   - Витягує 33 ключові точки (Landmarks).
   - Відправляє `Array<Point3D>` назад у Main Thread.

## Структура Payload
Кожна точка, яку повертає MediaPipe, має формат:
- `x`, `y` - нормалізовані координати екрана (від 0.0 до 1.0).
- `z` - орієнтовна глибина відносно стегон (в метрах).
- `visibility` - ймовірність (0.0 - 1.0) того, що точка реально видима в кадрі, а не перекрита.
